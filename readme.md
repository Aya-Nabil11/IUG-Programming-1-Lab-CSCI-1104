## 🧭 About This Repository

This is more than a class code archive — it is a **ready-to-use Java curriculum** for anyone who wants to **learn Java from scratch** or **teach Java** to beginners.

Everything is organized week by week and covers the complete journey from printing your first line of code to writing Object-Oriented programs. Each week includes:

- 💻 **Lab source code** — clean, well-commented Java examples written during sessions
- 📝 **Assignments** — real tasks assigned to students throughout the course
- 🧪 **Quizzes** — practice questions used to assess student understanding
- 🎬 **Recorded lectures** — linked YouTube videos for every topic
- 📄 **Final project** — a capstone spec to tie everything together

Whether you're a student reviewing material, a self-learner starting Java, or an instructor looking for structured teaching material — this repo has you covered.

## 📚 Full Curriculum — Week by Week

### 🟢 Week 0 — Introduction to Java
Your very first Java program. Learn how a Java file is structured, how to compile and run it, and how to print output to the console.

**Topics:** Java program structure, `System.out.println()`, comments, compilation basics

---

### 🟢 Week 1 — Variables, Data Types & Operators
The foundation of any program — storing and manipulating data.

**Topics:** Primitive types (`int`, `double`, `char`, `boolean`), variable declaration and initialization, arithmetic operators (`+`, `-`, `*`, `/`, `%`), assignment operators, increment/decrement (`++`, `--`), operator precedence, type casting

---

### 🟢 Week 2 — String Data Type
Working with text in Java using the powerful `String` class.

**Topics:** String declaration, concatenation, string methods — `length()`, `charAt()`, `substring()`, `indexOf()`, `toUpperCase()`, `toLowerCase()`, `trim()`, string comparison with `equals()` vs `==`

---

### 🟢 Week 3 — Scanner & User Input
Making your programs interactive by reading input from the user.

**Topics:** Importing and using `Scanner`, reading `int`, `double`, `String` input, `nextLine()` vs `next()`, escape sequences (`\n`, `\t`, `\"`, `\\`)

---

### 🟢 Week 4 — Flow Control
Teaching your program to make decisions.

**Topics:** `if`, `else if`, `else` statements, nested conditions, `switch-case`, logical operators (`&&`, `||`, `!`), relational operators

---

### 🟢 Week 7 & 8 — Loops
Repeating actions efficiently — one of the most important concepts in programming.

**Topics:** `for` loop, `while` loop, `do-while` loop, nested loops, `break` and `continue`, loop patterns and exercises

---

### 🟢 Week 10 & 11 — Arrays
Storing and working with collections of data.

**Topics:** Declaring and initializing arrays, accessing elements by index, iterating with loops, common algorithms (sum, average, min/max, search), multi-dimensional (2D) arrays, passing arrays to methods

---

### 🟢 Week 12 — Object-Oriented Programming (OOP)
Writing structured, reusable, real-world code.

**Topics:** Classes and objects, fields (attributes), methods, constructors, `this` keyword, encapsulation basics, creating and using instances

## 🚀 Getting Started

**Prerequisites:** Java JDK 8+ installed on your machine.

```bash
# 1. Clone the repository
git clone https://github.com/aya-alharazin/IUG-Programming-1-Lab-CSCI-1104.git

# 2. Navigate to a weekly folder
cd week01variables

# 3. Compile a Java file
javac JavaVariables.java

# 4. Run the program
java JavaVariables
```

> 💡 For a smoother experience, open the project in **IntelliJ IDEA**, **Eclipse**, or **VS Code** with the Java Extension Pack.

### Recommended Learning Path
1. Watch the YouTube lecture for the week 🎬
2. Open the corresponding folder and read the source files 📂
3. Run the code and experiment with changes 💻
4. Complete the assignment and quiz for that week 📝
5. Move on to the next week ➡️
